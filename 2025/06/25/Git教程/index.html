Git Learning

常用命令

创建仓库

git init 本地创建仓库（新建文件夹后进入文件夹路径输入命令）
git clone <url> 克隆github等远程仓库
文档中的.git目录为配置文件目录

查看仓库的状态

git status 查看仓库的状态
git diff 查看变更内容

跟踪和提交

git add . 添加所有文件目录下的文件到暂存区
git add <file> 添加指定文件到暂存区
git commit 提交到本地仓库（会跳入到修改说明界面）
git commit -m "<str>" 提交到本地仓库（直接提交修改和修改说明）

回退

git reset --soft <commit id> 回退但保留工作区和暂存区的修改
git reset --hard <commit id> 回退不保留工作区和暂存区的修改
git reset --mixed <commit id> 回退只保留工作区的修改

删除

git rm <file> 把文件从工作区和暂存区同时删除
git rm --cached <file> 把文件从暂存区删除，但保留在当前工作区中
git rm -r * 递归删除某个目录下的所有子目录和文件
删除后不要忘记提交

忽略

.gitignore文件用于忽略，将文件写入.gitignore中忽略

  1. 空行或者以#开头的行会被Git忽略。一般空行用于可读性的分隔，#一般用作注释
  2. 使用标准的Blob模式匹配，例如：
    *通配任意个字符，例如*.a忽略所有的.a文件
    ?匹配单个字符
    []表示匹配列表中的单个字符，比如：[abc]表示a/b/c
    **表示匹配任意的中间目录
    [0-9]表示任意一位数字
    [a-z]表示任意一位小写字母
    使用!可以取反

上传和拉取

git push 上传到远程仓库
git pull 远程仓库拉取到本地仓库
git pull <远程仓库名> <远程分支名>:<本地分支名> 拉取远程仓库内容

远程操作

git remote -v 查看仓库关联

分支

git branch 查看分支*标记为当前分支
git branch <branch> 创建分支
git switch <branch> 切换分支
git merge <branch> 合并分支
git branch -d <branch> 删除已合并的分支
git branch -D <branch> 删除未合并的分支
git log --oneline --graph --decorate --all 查看分支日志
